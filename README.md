## Задание 2
### Написать программу Hello World на C# с запуском на Unity.
Ход работы:
- Скачать Unity и авторизоваться.
- Скачать среду разработки для работы с C# (в моем случае Microsoft Visual Studio).
- Создать 3D проект.
- Добавить пустой GameObject.
- Написать скрипт компонента, который будет выводить в консоль "Hello World!".

```C#

using System.Collections;
using System.Collections.Generic;
using UnityEngine;

public class Helloworld : MonoBehaviour 
{
    void Start()
    {
        Debug.Log("Hello world!");
    }
}


```
- Повесить скрипт на объект и запустить проект.

## Задание 3
### Какую сущность(и) мы бы могли "обучить" ML-Agent-ом для того чтобы создать более качественный игровой опыт?
Сущность может выполнять задачи описанные ниже
- NPC. Поведение: Обучите NPC адаптивному поведению, например, чтобы они реагировали на действия игрока, изменяли свои стратегии в зависимости от ситуации или взаимодействовали друг с другом. Тактические действия: NPC могут учиться принимать более сложные тактические решения в бою или при выполнении различных задач.
- Сущность может определять модель поведения у различных врагов
- Искусственный интеллект врагов. Стратегии боя: Враги могут учиться адаптироваться к стилю игры игрока, меняя свои тактики. Сложность: Автоматическая настройка уровня сложности в зависимости от умений игрока.
